# IfdalTools
IfdalTools memudahkan orang lain untuk membantu hal hal sederhana seperti menghitung jarak kecepatan waktu, menghintung BMI, menghitung diskon harga, kalkulator dan menghitung PPN
# 🛠️ IfdalTools Pro
Website alat bantu edukatif yang dirancang untuk memudahkan perhitungan harian dan pembelajaran Informatika.

### 🚀 Fitur Utama:
* **🚗 Smart JKW:** Menghitung Jarak, Kecepatan, dan Waktu dengan konversi otomatis (Fisika).
* **⚖️ Kalkulator BMI:** Analisis berat badan ideal beserta saran target kesehatan.
* **🧾 Hitung PPN:** Simulasi perhitungan pajak 11% untuk belanja.
* **🏷️ Cek Diskon:** Menghitung harga akhir setelah potongan harga.
* **➕ Kalkulator:** Operasi matematika dasar yang cepat.

### 📖 Tujuan Proyek:
Dibuat sebagai bagian dari proyek pembelajaran **Informatika** dan pengembangan web menggunakan **Acode**.
